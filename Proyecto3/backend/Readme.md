# Rutas

| Ruta | Función |
| :---         |     :---      |
| ```/api/producto/``` | obtener lista de productos  |
| ```/api/producto/:idproducto```  | obtener información de un producto en especifico  |
| ```/api/producto/random/:cantidad```  | obtener ```cantidad``` de productos al azar  |
